# 🧠 AI Resume Analyzer

A modern, AI‑powered resume analysis tool that helps job seekers instantly see how well their resume matches a job description. It provides a detailed match score, ATS compatibility check, skill gap analysis, actionable suggestions, a downloadable PDF report, and even a tailored cover letter — all for free.

![Netflix-style dark UI](https://img.shields.io/badge/UI-Netflix%20Dark%20Theme-red) ![Free to use](https://img.shields.io/badge/Cost-Free-brightgreen) ![MIT License](https://img.shields.io/badge/License-MIT-yellow)

## 🚀 Features

- **AI Resume Parsing** – Extracts name, contact info, skills, work experience, and education using Groq’s LLaMA 3.1.
- **Smart Match Analysis** – Compares your resume against a job description, highlighting matching and missing skills.
- **Experience Level Check** – Determines if your years of experience meet the job’s requirements.
- **ATS Compatibility Scan** – A 7‑point checklist (file format, headings, keywords, etc.) that helps your resume pass automated filters.
- **Semantic Match Score** – A true cosine‑similarity score between your resume and the job description.
- **Actionable Suggestions** – AI‑generated tips to improve your resume for that specific job.
- **One‑Click PDF Report** – Download a beautifully formatted analysis report.
- **Cover Letter Generator** – Instantly creates a tailored cover letter based on your resume and the job description.
- **Beautiful Netflix‑Inspired UI** – Dark theme, glass‑morphism cards, and smooth animations.

| **Backend**    | FastAPI (Python 3.11+), Uvicorn, python‑multipart |
| **AI / NLP**   | Groq Cloud (LLaMA 3.1 8B), sentence‑transformers (all‑MiniLM‑L6‑v2) |
| **Text Parsing** | pdfplumber, python‑docx, unstructured (optional) |

## 🛠️ Getting Started (Local Development)

### Prerequisites
- Python 3.10+
- Node.js 18+
- Git
- A [Groq API key](https://console.groq.com) (free)
- A [Supabase account](https://supabase.com) (free) with a project

### 1. Clone the repository
```bash
git clone https://github.com/your-username/resume-analyzer.git
cd resume-analyzer
2. Backend setup
bash
cd backend
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate
pip install -r requirements.txt
Create a .env file inside backend/:

  created_at timestamp with time zone default now()

🌐 Deployment
Backend (Render)
Push your code to GitHub.

On Render, create a new Web Service, connect the repo.

Root Directory: backend

Build Command: bash build.sh (or pip install -r requirements.txt)

Start Command: uvicorn main:app --host 0.0.0.0 --port $PORT

Add environment variables: GROQ_API_KEY, SUPABASE_URL, SUPABASE_KEY, PYTHON_VERSION=3.11.13.

Deploy and note the URL (e.g., https://resume-backend.onrender.com).

To keep it awake, set up a free cron job at cron-job.org that pings the / endpoint every 10 minutes.

Frontend (Vercel)
Import your GitHub repo on Vercel.

Root Directory: frontend

Add environment variable: BACKEND_URL = your Render backend URL.

Deploy. You’ll get a public URL like https://resume-analyzer.vercel.app.

Drag‑and‑drop your resume (PDF or DOCX) and paste a job description.

Explore the results:

ATS compatibility checklist

Improvement suggestions

Download the PDF report or generate a tailored cover letter.
